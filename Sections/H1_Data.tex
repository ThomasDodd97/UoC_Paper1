\section{Data availability}
Custom software developed for the purpose of execution of optimisation
 routines is available under the `BO4BT' project lodged in github at
 the following address: https://github.com/ThomasDodd97/BO4BT

The latex project associated with this paper has been made available
 under the `UoC\_Paper1' project lodged in github at the following 
 address: https://github.com/ThomasDodd97/UoC\_Paper1

Further to this, all the data resulting from execution of the custom
 software has been tabulated, analysed, and visualised in the 
 `sandbox' directory that exists within the `UoC\_Paper1' project.
